Nosimtrenky.cz - Zásilkovna
=================

Formulář pro vybrání pobočky Zásilkovny a odeslání údajů k zásilce přes API do Zásilkovny.

Požadavky
------------

- Nette 3.0 
- PHP 7.2+

Instalace
------------

V adesáři ve kterém máte projekt spusťte:

	composer install
	

Adresáře `temp/` a `log/` musí mít práva pro zápis.
